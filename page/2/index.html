2015年工作计划

#####计划

1.完成毕业论文成功毕业。
2.从事一份好的自己喜欢的工作！
3.学习弹钢琴。
4.完成一次旅行。
5.每月至少发两次博客。
6.构建MyBlog主页。
7.完成MyList项目。
8….

#####年度目标

(进阶)对于常用的设计模式、内存管理、Blocks 的使用、图像操作、网络请求和管理、多线程达到比较熟悉。对于 CALayer、Animation、UIScrollView、UITableView、UICollectionView、ViewController Container 则非常熟悉，对「非常熟悉」的定义是：不打开 Xcode，脑子里就能把相应的知识点复述出来 80% ，比如这个类有哪些方法，Delegate / DataSource 有哪些方法，怎么使用，如果要实现某个效果，应该怎么做（好吧， UICollectionView 除外）。

(高级)了解各种 Core 开头的 Framework，工具也能熟练使用，「正经的代码」写过数万行，可能天天在翻 Dash。如果别人让你实现某个功能，能在较短的时间内给出不错的实现方案，并且足够细致，甚至精细到如何使用 Core Graphic 去画某个图像。
