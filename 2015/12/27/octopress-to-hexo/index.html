octopress迁移至hexo

  • hexo

根据官方README.md配置

$ npm install hexo-cli -g

下周基本配置导blog文件。

$ hexo init blog
$ cd blog
$ npm install

之后就OK了

$ hexo server

启动服务，我这里的地址是http://0.0.0.0:4000/,个人觉得相对于octopress来说配置简单了许多.

$ hexo new "Hello Hexo"
$ hexo generate 
$ hexo deploy

链接git

安装hexo-deployer-git

$ npm install hexo-deployer-git --save

_config.yml中添加下面的东西

deploy:
    type: git 
      repo: <repository url> // GitHub repository URL
      branch: [branch] // Branch name
      message: [message] // Customize commit message

更换皮肤

来自于litten的hexo-theme-yilia,替换方法也很简单。

$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia

配置
修改hexo根目录下的 _config.yml ： theme: yilia

更新

cd themes/yilia
git pull
